# Space Invaders Game

This is a classic Space Invaders game implemented in HTML5 and JavaScript with high-quality graphics using SVG-based sprites. 

## Features
- Smooth player and invader animations
- SVG-based graphics for crisp visuals
- Score tracking
- Game over detection
- Responsive controls

## How to Play
- Use **Left** and **Right** arrow keys to move the player
- Press **Space** to shoot
- Destroy all invaders to advance to the next wave
- Avoid letting invaders reach your ship

## Customization
- Replace SVG data URLs in `space-invaders.js` with your own images for even higher quality
- Adjust game parameters for difficulty and style

## To Run
Just open `index.html` in your browser.
